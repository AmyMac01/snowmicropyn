Welcome to snowmicropyn
=======================

A warmly welcome to the documentation of snowmicropyn, a python package
to read and post-process data recorded by SnowMicroPen, a snow
penetrometer developed at SLF.

Installation
------------

By using pip_, the installation of snowmicropyn is a simple task. In a
Terminal, execute the following command:

.. code-block:: console

    pip install snowmicropyn

This will install the latest version available on PyPI_. In case you
want to install a certain version, specify its version number as follows

.. code-block:: console

    pip install snowmicropyn==0.1.3


.. _PyPI: https://pypi.org/
.. _pip: https://pip.pypa.io/en/stable/quickstart/
